Axios(http库)

Axios 是一个基于 promise(?) 的 HTTP 库 ，可以用在浏览器和 node.js 中。

  • 特点:

    从浏览器中创建 XMLHttpRequests
    从 node.js 创建 http 请求
    支持 Promise API
    拦截请求和响应
    转换请求数据和响应数据
    取消请求
    自动转换 JSON 数据
    客户端支持防御 XSRF

node.js

  • web服务器：接受http请求，返回相关的web页面

（不仅实现应用，而且实现整个http服务器）
